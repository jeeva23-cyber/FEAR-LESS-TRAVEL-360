 FearLess Travel 360 is an AI-powered tourist safety web application that combines GPS tracking, Google Maps, Geo-Fencing, SOS emergency alerts, AI chatbot assistance, and QR-based digital identity. It helps travelers stay safe with real-time monitoring, route guidance, nearby emergency services, and an admin dashboard for incident management.
# 🌍 FearLess Travel 360

**AI-Powered Tourist Safety Monitoring & Incident Response System**

FearLess Travel 360 is a smart web application designed to improve tourist safety using AI and real-time location services. It provides GPS tracking, Google Maps integration, Geo-Fencing, SOS emergency alerts, AI chatbot assistance, QR-based digital identity, and an admin dashboard for monitoring and incident management.

## 🚀 Features

- 🔐 Tourist Registration & Login
- 📍 Live GPS Tracking
- 🛰️ Google Maps (Satellite, Roadmap, Hybrid & Terrain)
- 🚨 SOS Emergency Alert
- 🛡️ Geo-Fencing
- 🤖 AI Chatbot
- 🆔 QR Code Digital Identity
- 🏥 Nearby Hospitals, Police Stations, Hotels & More
- 📊 Admin Dashboard
- 📱 SMS Notification
- ☁️ Firebase Authentication & Firestore

## 🛠️ Technologies Used
